# My NixOS Journey

I briefly checked out NixOS some time ago but quickly switched back to Ubuntu lol.

My dotfiles: [github](https://github.com/pierresquarra/nix-config)

## Some of the key components of my configuration:

- **Window Manager**: Hyprland
- **Shell**: Bash
- **Terminal**: Kitty
- **Panel**: Waybar
- **Notification Daemon**: Dunst
- **Launcher**: Wofi
- **File Manager**: Thunar
